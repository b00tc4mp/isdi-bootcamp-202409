## Functional

### Use Cases

Regular (User)
- publish a post
- delete a post
- like / unlike a post
- add / remove comment
- follow / unfollow user
- chat with user
- report a user / post / comment

Moderator (User)
- view reporting list
- view reporting detail
- ban user / post / comment
